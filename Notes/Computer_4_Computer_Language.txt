# 2️⃣ LANGUAGE TRANSLATORS - DEEP DIVE

## 🔄 Types of Translators - Detailed

### 1. Assembler
- **Function:** Assembly → Machine Code
- **Mapping:** 1:1 (One mnemonic = One machine instruction)
- **Output:** Object file (.obj)

### 2. Compiler
- **Function:** HLL → Machine Code
- **Mapping:** 1:N (One HLL statement = Multiple machine instructions)
- **Output:** Executable file (.exe)
- **Phases:** Lexical → Syntax → Semantic → Intermediate → Optimization → Code Generation

### 3. Interpreter
- **Function:** HLL → Execution (line by line)
- **No object code generated**
- **Better for debugging**

### 4. Cross-Compiler
- **Definition:** Compiler that runs on one platform but generates code for another
- **Example:** Compiling on Windows for Linux

### 5. Incremental Compiler
- **Definition:** Compiles only changed portions
- **Example:** Java with JIT (Just-In-Time)

### 6. Decompiler
- **Definition:** Machine code → HLL (reverse engineering)

---

**Rules: "char → int → long → float → double → long double"**


```
     long double
         ↑
       double
         ↑
       float
         ↑
      long
         ↑
      unsigned
         ↑
       int
         ↑
      short/char
```

## 🔹 Parameter Passing - Detailed

### Call by Value
- Copy of data passed
- Original unchanged
- Default in C

### Call by Reference (using pointers)
- Address passed
- Original can be modified
- Uses `*` and `&`

```c
// Swap using call by reference
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int x=5, y=10;
swap(&x, &y);  // Now x=10, y=5
```

```c
void hanoi(int n, char from, char to, char aux) {
    if(n == 1) {

    printf("Move %d from %c to %c\n", n, from, to);
    hanoi(n-1, aux, to, from);
}
```

# 1️⃣7️⃣ STRUCTURES & UNIONS

## 🔹 Structure vs Union - Critical Difference

| Feature | Structure | Union |
|---------|-----------|-------|
| **Memory** | Sum of all members | Size of largest member |
| **Member access** | All members accessible | Only one member at a time |
| **Storage** | Separate space for each | Shared space |
| **Initialization** | Can initialize all | Can initialize only first |
| **Use case** | Group related data | Save memory, variant data |

---

```c
struct Example {
    char a;    // 1 byte + 3 padding
    int b;     // 4 bytes
    char c;    // 1 byte + 3 padding
};  // Total = 12 bytes (not 1+4+1=6)

// To avoid padding (not always possible)
#pragma pack(1)  // Disable padding
struct Packed {
    char a;
    int b;
    char c;
};  // Now = 6 bytes
```

# 1️⃣8️⃣ FILE HANDLING (NEW - IMPORTANT)

## 🔹 File Operations

### File Modes

| Mode | Meaning | If file exists | If file doesn't exist |
|------|---------|----------------|----------------------|
| `"r"` | Read | Read from start | Error |
| `"w"` | Write | Overwrite | Create new |
| `"a"` | Append | Append to end | Create new |
| `"r+"` | Read/Write | Read/Write from start | Error |
| `"w+"` | Read/Write | Overwrite | Create new |
| `"a+"` | Read/Append | Read/Append | Create new |
| `"rb"` | Read binary | Read binary | Error |
| `"wb"` | Write binary | Overwrite binary | Create new |

## 🔹 File Functions

```c
FILE *fp;

// Open file
fp = fopen("file.txt", "r");
if(fp == NULL) {
    printf("Cannot open file");
    exit(1);
}

// Character I/O
fgetc(fp);      // Read character
fputc('A', fp); // Write character

// String I/O
fgets(str, 50, fp);  // Read string
fputs(str, fp);       // Write string

// Formatted I/O
fscanf(fp, "%d", &x);
fprintf(fp, "%d", x);

// Binary I/O
fread(&data, sizeof(data), 1, fp);
fwrite(&data, sizeof(data), 1, fp);

// Close file
fclose(fp);
```

## 🎯 EXAM TRICK: Macro vs Function

```c
#define SQUARE(x) x*x
// Problem: SQUARE(2+3) expands to 2+3*2+3 = 11 (not 25)

#define CORRECT_SQUARE(x) ((x)*(x))
// Now CORRECT_SQUARE(2+3) = ((2+3)*(2+3)) = 25

// Macros are faster but no type checking
// Functions are slower but type-safe
```

```c
int main(int argc, char *argv[]) {
```

# 2️⃣4️⃣ COMMON ERRORS & DEBUGGING (NEW - IMPORTANT)

## 🔹 Compilation Errors

| Error Type | Example | Fix |
|------------|---------|-----|
| **Syntax Error** | Missing semicolon | Add `;` |
| **Type Mismatch** | Assigning float to int* | Use correct type |
| **Undeclared Variable** | `printf("%d", x);` without declaring x | Declare `int x;` |
| **Missing Header** | Using printf without #include | Add `#include <stdio.h>` |

---

## 🔹 Runtime Errors

| Error | Cause | Detection |
|-------|-------|-----------|
| **Segmentation Fault** | Accessing invalid memory | Check pointers, array bounds |
| **Stack Overflow** | Infinite recursion | Check base case |
| **Division by Zero** | Dividing by 0 | Check denominator |
| **Buffer Overflow** | Writing beyond array | Use bounds checking |

---

## 🔹 Logical Errors

```c
// Wrong: = instead of ==
if(x = 5)  // Always true, assigns 5 to x

// Correct:
if(x == 5)

// Off-by-one errors
for(i = 1; i <= 10; i++)  // Correct (10 iterations)
for(i = 0; i < 10; i++)    // Also correct (10 iterations)
for(i = 0; i <= 10; i++)   // Wrong! 11 iterations
```

```
C - char, const, continue
I - int, if
```

```
C - char (1)
I - int (2/4)
```

```
P - Primary
U - Unary
M - Multiplicative
A - Additive
S - Shift

R - Relational
E - Equality
L - Logical AND
A - Logical OR
C - Conditional
```

### Storage Classes - "ARSE"
```
A - auto
R - register
S - static
E - extern
```

### File Modes - "RWA+"

## 📝 Last Minute Tips

1. **Read questions carefully** - look for trick words like "not", "except", "always"
2. **Check operator precedence** - most common mistakes

4. **Strings end with '\0'** - always account for it
5. **Pointers size is same** - regardless of data type
6. **Static variables retain values** - across function calls
7. **Union uses largest member's memory** - structure uses sum
8. **Function name without () is address** - pointer to function
